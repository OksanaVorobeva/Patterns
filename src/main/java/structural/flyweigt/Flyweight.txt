Шаблон: Приспособленец (Flyweight)

Цель:

Поддержка множества мелких обьектов.

Для чего используется:

Использует разделение для того, чтобы поддержать много мелких обьектов.

Пример использования:

-когда используется большое число обьектов;
-большую часть состояния обьектов можно вынести наружу;
-приложение не зависит от идентичности обьекта.
