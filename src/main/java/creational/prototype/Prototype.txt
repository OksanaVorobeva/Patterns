Шаблон: Прототип (Prototype)

Цель:

Определить вид создоваемых обьетов с помощью экземпляра-прототипа и создавать новые обьекты,
копируя этот прототип.

Для чего используеться:

Для создания копий заданного обьекта.

Пример использования:

-классы, экземпляры которых необходимо создавать определяються во время выполнения программы;
-для избежания построения иерархии классов, фабрик или параллельных иерархий классов;
- экземпляры класса могут находиться в одном из немногих возможных состояний;
