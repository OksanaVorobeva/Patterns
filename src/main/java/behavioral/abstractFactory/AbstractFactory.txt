Шаблон:  Абстрактная фабрика(Abstract Factory)

Цель:

Создание интерфейса, для создания множества взаимосвязанных или взаимнозависимых обьетов,
без жесткой привязки к конкретным классам.

Для чего используется:

Для создания множеств взаимосвязанных обьетов.

Примет использования:

- Система не должна зависить от методов создания, компоновки и представление входящих в нее обьетов;
-входящие взаимосвязанные обьекты должны использоваться вместе;
-система длжна конфигурироваться одним из множеств обьектов,из которых она состоит;
- нам необходимо представить множество обьектов, раскрывая только их интерфейсы на реализацию.
