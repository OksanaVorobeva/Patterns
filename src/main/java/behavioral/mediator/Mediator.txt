Шаблон: Посредник (Mediator)

Цель:

Инкапсуляция способа взаимодествия множества обьектов

Для чего используется:

Для определения обьекта,который инкапсулирует способ взаимодейсвия множества обьектов и обеспечения
слабой связи между этим обьектами.

Пример использования:

-связи между обьектами сложно четко определены;
-нельзя повторно использовать обьект, так как он обмениявается информацией с другими обьектами;
-поведение, распределенное между несколькими классами должно легко настраиваться без создания подклассов.
