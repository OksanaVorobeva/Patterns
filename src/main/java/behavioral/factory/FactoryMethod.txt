Шаблон: Фабрика(Factory Method)

Цель:

Создание интерфейса, который создает обьект. При этом, выбор того, экземпляра какого классса создавать
остается за классами, которые имплементируют данный интерфейс.

Для чего используется:

Для делегирования создания экземпляров, другому классу.

Пример использования:

- заранее неизвестно, экземпляры, какого класса нужно будет создавать;
-класс спроетирован такми образом, что создаваемые им обьекты имеют свойства определенного класса.
